## 🚀 部署到公网（3 种方式）

### 方式 1：Netlify Drop（最简单，推荐）

1. **访问**：https://app.netlify.com/drop
2. **拖拽**：把 `toeic-game/dist` 文件夹拖到页面
3. **获得链接**：立即生成公网链接！
4. **分享**：发送链接给同事

**优点**：
- ✅ 无需注册登录
- ✅ 1 分钟搞定
- ✅ 永久免费
- ✅ 支持 HTTPS
- ✅ 全球 CDN 加速

### 方式 2：Vercel 部署

1. **访问**：https://vercel.com/new
2. **登录**：使用 GitHub 账号
3. **导入**：选择 `toeic-game` 仓库
4. **部署**：点击 "Deploy"
5. **获得链接**：`https://toeic-game.vercel.app`

**优点**：
- ✅ 自动同步（每次 git push 自动更新）
- ✅ 永久免费
- ✅ 更快的 CDN

### 方式 3：本地测试（临时）

```bash
# 进入 dist 目录
cd toeic-game/dist

# 启动本地服务器
npx http-server -p 8080

# 访问
# http://localhost:8080
# 或 http://你的 IP:8080
```

## 📱 手机访问

### 部署成功后

1. **打开手机浏览器**
2. **输入公网链接**
3. **首次访问**：点击 "Click to Continue"（如果有）
4. **开始游戏**！

### 如果手机访问不了

**检查项**：
- [ ] 电脑是否开机
- [ ] 本地服务器是否运行
- [ ] 防火墙是否允许
- [ ] 手机和电脑是否同一网络

**解决方案**：
- 使用 Netlify Drop 部署（推荐）
- 或使用 Vercel 部署
- 这样链接永久有效，不依赖电脑

## 🎯 游戏规则

### 基本玩法
1. **点击卡牌**翻开
2. **找到配对**：单词 ↔ 中文释义
3. **匹配成功**：消除并得 10 分
4. **全部消除**：通关

### 评分标准
- ⭐⭐⭐：10 步以内完成
- ⭐⭐：11-15 步完成
- ⭐：15 步以上完成

### 游戏技巧
- 记住牌的位置
- 先翻边缘的牌
- 步数越少，星级越高

## 📞 常见问题

### Q1: 单词本打不开？
**A**: 点击首页"单词本"按钮，可以搜索和导出单词。

### Q2: 手机上看不了？
**A**: 
1. 确保已部署到公网（Netlify/Vercel）
2. 使用手机浏览器访问
3. 清除浏览器缓存

### Q3: 如何分享同事？
**A**: 
1. 部署到 Netlify 获得公网链接
2. 发送链接给同事
3. 同事直接打开就能玩

### Q4: 如何更新单词？
**A**: 
1. 修改 `toeic-game/src/toeicWords.js`
2. 运行 `npm run build`
3. 重新部署 `dist` 文件夹

### Q5: 如何添加更多单词？
**A**: 
1. 打开 `toeic-game/src/toeicWords.js`
2. 按格式添加单词：
```javascript
{ word: 'word', phonetic: '/phonetic/', meaning: '意思' },
```
3. 重新构建和部署
