# 🚀 自动部署设置指南

## 📋 完成这些步骤后，每次保存代码会自动同步到公网！

---

## 方式 1：GitHub + Vercel（推荐，最简单）

### 第 1 步：上传代码到 GitHub

1. **访问** https://github.com/new
2. **创建仓库**：
   - 仓库名：`toeic-game`
   - 设为公开（Public）
   - 点击 "Create repository"

3. **上传代码**（在 toeic-game 文件夹执行）：
```bash
cd toeic-game
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/toeic-game.git
git push -u origin main
```

### 第 2 步：连接 Vercel

1. **访问** https://vercel.com/new
2. **登录**：使用 GitHub 账号登录
3. **导入项目**：
   - 点击 "Import Git Repository"
   - 选择 `toeic-game` 仓库
   - 点击 "Import"

4. **部署**：
   - 点击 "Deploy"
   - 等待部署完成（约 1-2 分钟）
   - 获得公网链接！

### ✅ 自动同步效果

从此以后：
- ✅ 每次 push 到 GitHub
- ✅ Vercel 自动构建
- ✅ 自动部署到公网
- ✅ 链接不变，内容自动更新！

---

## 方式 2：GitHub + GitHub Pages（完全免费）

### 第 1 步：同上，上传到 GitHub

### 第 2 步：启用 GitHub Pages

1. **访问** 你的仓库页面
2. **Settings** → **Pages**
3. **Source**：选择 `gh-pages` 分支
4. **Save**

### 第 3 步：获得链接

链接格式：
```
https://你的用户名.github.io/toeic-game
```

### ✅ 自动同步效果

- ✅ 每次 push 到 GitHub
- ✅ GitHub Actions 自动运行
- ✅ 自动构建并部署
- ✅ 约 2-3 分钟更新完成

## 📝 日常工作流程

### 修改代码后

```bash
# 1. 保存代码
# 2. 提交到 Git
git add .
git commit -m "修改内容描述"

# 3. 推送到 GitHub
git push

# 4. 等待自动部署（1-2 分钟）
# 5. 刷新公网链接查看更新！
```

### 或者使用 VS Code

1. 保存文件（Ctrl+S）
2. 点击左侧源代码管理图标
3. 输入提交信息
4. 点击 "同步更改"
5. 完成！
